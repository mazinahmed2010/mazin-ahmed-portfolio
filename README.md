# 💼 Mazin Ahmed Portfolio

A modern **FAANG-style personal portfolio** built with **Vue 3 + Vite**, showcasing my background as a **Full Stack Web Developer** and **Civil Engineer**.

---

## 🌐 Live Demo

https://mazin-ahmed-portfolio.netlify.app

## ✨ Features

- 🎨 Modern FAANG-style UI
- 🌙 Dark / Light Mode
- ⚡ Smooth animations with **AOS** and **GSAP**
- 📱 Fully responsive mobile-first design
- 🧠 Dynamic project filtering
- 📄 Download CV button
- 🧭 Floating professional navbar
- 📊 Scroll progress bar
- ⏳ Loader screen
- ⬆️ Back To Top button
- 🔗 Social media footer with brand icons
